About Playcert

This project was created with the purpose of showing the concerts that will happen during the following week around the current location, or around a chosen location, as well as providing a spotify playlist based on these concerts.

The source code for this project is freely available at github: https://github.com/Shemahmforash/playcert so feel free to fork it.

If you have any doubt or suggestion, don't hesitate in reaching me through twitter: @shemahmforash.
